# Koukaton Up

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 主人公(工科トン)をどんどん上に登らせるゲーム
* 参考URL：[Only Up!](https://store.steampowered.com/app/2842590/Only_Up/?l=japanese)
         ：[Square On Top: Jump](https://apps.apple.com/app/id1502021368)

## ゲームの遊び方
* キーを使って床をどんどん登って上に行く
* 落ちたら床があるところまで落ちる
* 面白い床もあるよ!

## ゲームの実装
![alt text](gamegamen-1.png)
### 共通基本機能
* 矢印でジャンプの方向決めてスペースの押す長さでジャンプ力を決める
* 壁に当たると跳ね返る

### 分担追加機能
* 1_グラフィック（画像）変更[ゆら],
* 2_当たり判定の厳密化（頭ぶつけ・すり抜け防止）[うりゅう],
* 3_ステージギミック追加[そら],
* 4_音楽[こうへい],
* 5_左上のUI（高さ・情報の表示）[こな]

### メモ
* クラス内の変数は，すべて，「get_変数名」という名前のメソッドを介してアクセスするように設計してある
* すべてのクラスに関係する関数は，クラスの外で定義してある
